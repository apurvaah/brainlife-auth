Usage: 

* List Users
./auth.js listuser 

* register new user
./auth.js newuser --username hayashis --fullname "Soichi Hayashi" --email "hayashis@iu.edu"

* Issue new JWT token
./auth.js issue --scopes '{ "sca": ["user"] }' --sub 'test_service' [--out test.jwt] [--key test.key]

* Reset password
./auth.js setpass --username hayashis --password "password#123"

* Modify (set/add/del) user scopes
./auth.js modscope --username hayashis --set '{"common": ["user", "admin"]}'
./auth.js modscope --username hayashis --add '{"common": ["user", "admin"]}'
./auth.js modscope --username hayashis --del '{"common": ["user", "admin"]}'
